\# AOC POC Instructions



\## Goal

Build a mocked Autonomous Multi-Agent Operation Center for telecom OSS/BSS infrastructure.



\## Important Constraints

\- This is a mock POC.

\- Do not build real SSH, database, network, or cloud integrations.

\- Use local folders, JSON, YAML, and log files to simulate infrastructure.

\- The main objective is to showcase Agentic AI behavior.

\- Keep code simple, modular, and easy to demonstrate.

\- Do not over-engineer.

\- Complete only the requested phase.

\- Do not start future phases without approval.

\- Add tests for the implemented phase.
